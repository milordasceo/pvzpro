# Эндпоинты (будет дополнено)

- auth:
  - POST /auth/login — вход (моки)
- employees:
  - GET /employees — список сотрудников (моки)
  - GET /employees/{id} — детали (моки)
- pvz:
  - GET /pvz — список ПВЗ (моки)
  - GET /pvz/{id} — детали ПВЗ (моки)

> Примечание: пока используем моки; реальные контракты будут добавлены после интеграции.
